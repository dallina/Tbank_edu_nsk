# Tbank_edu_nsk
Этот репозиторий представляет из себя задания для отбора на смену от Т-банк Образования в Новосибирске.
task 1 - код для первого задания.
На вход поступают 2 числа через пробел, начало и конец диапазона соотвественно.
task 2 - код для второго задания.
В коде использовались модули os и json для сохранения всех изменений в файл. 
Каждая книга - элемент словаря, которому присущи название, автор, жанр, год издания, краткое описание. В библиотеку новая книга добавляется с помощью adds()
Также функцией get_id() каждой книге присуждается индвидуальный ID. 
Функционал сортировки книг обспечивается browse_collection(). Поиск - find_books()
edit_book_record() - позволяет изменять статус книг: прочитана/ не прочитана, а также добавлять/убирать в/из Избранное.
